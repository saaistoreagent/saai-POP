# POP 메이커 — 편의점 POP 자동 생성기

## 1. 프로젝트 개요

편의점 점주가 상품 정보만 입력하면 A4 인쇄용 POP(Point of Purchase)를 자동으로 만들어주는 웹 앱.
행사 포스터, 행사 배지, 선반 가격표, 띠지 4가지 POP를 지원하며, AI 기반 이미지 합성과 Canvas 템플릿 렌더링을 조합해 빠르고 정확한 결과물을 제공한다.

---

## 2. 문제 정의

### 현재 편의점 POP 제작의 어려움
- **프랜차이즈 본사 POP**: 획일적이고, 자체 행사나 지역 맞춤 POP는 지원 안 됨
- **수작업 POP**: 손글씨로 직접 쓰면 시간이 오래 걸리고 퀄리티가 낮음
- **디자인 툴 (Canva 등)**: 비전문가에게 진입 장벽이 높고, 편의점 POP 특화 템플릿이 없음
- **인쇄소 외주**: 비용이 높고 납기가 길어 즉시 대응 불가

### 타깃 사용자
- 편의점 점주 (프랜차이즈 + 개인)
- 소형 마트, 무인점포, 베이커리 등 소매업 자영업자
- 편의점 본사 마케팅 담당자 (시안 빠르게 만들 때)

---

## 3. 솔루션

### 핵심 가치
> **"상품 정보만 입력하면 인쇄 가능한 POP가 나온다"**

- 3단계 위저드로 필요한 정보만 순서대로 입력
- AI가 문구를 자동 제안하고, 배경 이미지를 생성
- A4 용지에 바로 인쇄 가능한 PNG 다운로드
- 결과물이 마음에 안 들면 자연어로 수정 요청

### 차별점 (vs Canva, vs ChatGPT에 직접 요청)
| 항목 | Canva | ChatGPT/Gemini 직접 | POP 메이커 |
|---|---|---|---|
| 편의점 POP 특화 | X | X | O |
| A4 인쇄 포맷 보장 | 수동 설정 | 보장 안 됨 | 자동 |
| 절취선/그리드 레이아웃 | 직접 제작 | 안 됨 | 자동 (배지 15개, 가격표 8개 등) |
| 행사 배지 (1+1 등) | 직접 디자인 | 매번 프롬프트 | 원클릭 |
| 한글 정확도 | 완벽 | 깨질 수 있음 | Canvas로 100% 정확 |
| 진입 장벽 | 높음 | 높음 (프롬프트) | 낮음 (위저드) |

## 4. POP 종류별 상세

### 4-1. 행사 포스터
- **용도**: 냉장고, 벽면, 출입문에 붙이는 큰 포스터
- **출력**: A4 한 장에 1장
- **생성 방식**: Gemini AI (한글 텍스트 직접 포함)
- **특징**:
  - 상품 사진 자동 검색 또는 직접 업로드
  - 분위기 프리셋 (맥주/음료, 라면/분식, 과자/디저트, 도시락/샐러드, 할인/특가, 프리미엄)
  - AI 문구 다듬기 (인라인, 원본 의미 유지)
  - 생성 후 자연어 수정 ("글자 더 크게", "배경 차갑게" 등)
- **비용**: Gemini API 호출당 약 50~70원

### 4-2. 행사 배지
- **용도**: 상품 옆에 붙이는 작은 행사 표시 딱지
- **출력**: A4 한 장에 세로 15개 (3x5) / 가로 12개 (4x3)
- **생성 방식**: Canvas 템플릿 (서버 렌더링)
- **특징**:
  - 1+1, 2+1, 3+1, 덤증정 등 프리셋
  - 직접 입력 (줄바꿈 가능)
  - 배경색 선택 (8종)
  - 절취선 자동
- **비용**: 무료

### 4-3. 선반 가격표
- **용도**: 선반에 끼워 넣는 상품별 가격표
- **출력**: A4 한 장에 세로 8개 (2x4) / 가로 6개 (2x3)
- **생성 방식**: Canvas 템플릿 (서버 렌더링)
- **특징**:
  - 상품 사진 ON/OFF 토글
  - 사진 ON: 상품 사진 자동 검색 + 카드 상단에 합성
  - 강조 문구 + 상품명 + 정상가(취소선) + 판매가
  - 상품 없이 문구만으로도 생성 가능
- **비용**: 무료

### 4-4. 띠지
- **용도**: 선반 앞면에 길게 붙이는 가로 띠 배너
- **출력**: A4 한 장에 세로 4줄 / 가로 3줄
- **생성 방식**: Canvas 템플릿 (서버 렌더링)
- **특징**: 선반 가격표와 동일한 옵션
- **비용**: 무료

## 5. 사용자 플로우

### 공통 플로우
```
[랜딩] → POP 종류 카드 선택
  ↓
[위저드 3단계] → 정보 입력 (종류별 다름)
  ↓
[결과 화면] → PNG 다운로드 + 수정 요청 (선택)
```

### 포스터 위저드
```
Step 1: 상품 정보 (상품명, 가격, 정상가)
  ↓
Step 2: 강조 문구 + 분위기
  - 프리셋 선택 or 직접 입력
  - AI 문구 다듬기 (선택, 인라인)
  - 분위기 프리셋 6종 or 직접 입력
  ↓
Step 3: 사진 확인 + 용지 방향
  - 상품 사진 자동 검색 (DuckDuckGo)
  - 직접 올리기 / 다른 이미지 찾기 / AI가 그리기
  - 용지 방향 (세로/가로)
  ↓
[생성] → Gemini AI
  ↓
[결과] → 수정 채팅 가능
```

### 선반/띠지 위저드
```
Step 1: POP에 뭘 넣을까요?
  - 강조 문구 (프리셋 + 직접 입력, 줄바꿈 가능)
  - 상품 사진 넣기 ON/OFF 토글
  ↓
Step 2: 상품 정보 (선택)
  - 상품명, 판매가, 정상가
  - 상품 없이 문구만 가능
  ↓
Step 3: 사진 확인 + 용지 방향 + 배경색
  ↓
[생성] → Canvas 즉시 렌더
```

### 배지 위저드
```
Step 1: 배지 문구 (프리셋 + 직접 입력)
  ↓
Step 2: 배경색 (8종)
  ↓
Step 3: 용지 방향
  ↓
[생성] → Canvas 즉시 렌더
```

## 6. 기술 아키텍처

### 프론트엔드
- **프레임워크**: Next.js 15 (App Router)
- **스타일**: Tailwind CSS 4
- **UI**: 모바일 앱 스타일 (max-width 440px, 앱 셸)
- **상태 관리**: React useState (서버 상태 없음)

### 백엔드 (Next.js API Routes)
| 엔드포인트 | 역할 | 사용 모델 |
|---|---|---|
| `/api/pop/generate` | POP 생성 (포스터: Gemini, 나머지: Canvas) | Gemini 3.1 Flash Image Preview / @napi-rs/canvas |
| `/api/pop/refine` | 결과 수정 (image-to-image) | Gemini 3.1 Flash Image Preview |
| `/api/pop/suggest` | 문구 AI 다듬기 | Gemini 2.5 Flash |
| `/api/search-image` | 상품 사진 검색 | DuckDuckGo 이미지 검색 |
| `/api/proxy-image` | 외부 이미지 프록시 (CORS 우회) | - |
| `/api/example/[type]` | 랜딩 카드 예시 이미지 | Canvas / Gemini |

### 이미지 렌더링
```
포스터 → Gemini API (embedKoreanText 모드)
  - 한글 텍스트를 Gemini가 직접 이미지에 렌더
  - 상품 사진 inline 전달
  - 수정 요청: image-to-image로 반복 수정

배지/선반/띠지 → @napi-rs/canvas (서버사이드)
  - Canvas 2D API로 직접 드로잉
  - 폰트: DoHyeon, Jua (TTF)
  - 절취선, 그리드, 배지, 가격 취소선 등 정밀 레이아웃
  - 사진 ON 시 product photo를 카드에 합성
```

### 외부 서비스
| 서비스 | 용도 | 비용 |
|---|---|---|
| Gemini 3.1 Flash Image Preview | 포스터 이미지 생성 + 수정 | ~50-70원/회 |
| Gemini 2.5 Flash | 문구 다듬기 | 무료 (일 1500회) / 유료 ~0.05원/회 |
| DuckDuckGo | 상품 사진 자동 검색 | 무료 |
