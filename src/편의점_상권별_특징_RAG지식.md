# 편의점 상권별 매출 패턴 — AI 판단용 지식 베이스

> **데이터 출처**: CU 12개 매장 × 1년치 (2024-06 ~ 2025-06) POS 데이터 4M 행
> **분석 방법**: 객수 1,000명당 판매수량 (매장 규모 영향 제거) → 12매장 평균 대비 배율
> **마지막 업데이트**: 2026-04-15
> **신뢰도**: 카테고리 단위 패턴은 신뢰 가능 (n=수만~수십만 거래). 개별 트렌드 상품 검증은 표본 작음.

## 사용 가이드 (AI에게)

이 문서를 읽고 다음 작업에 활용하세요:
1. **매장 정보(주소·주변 상권)를 보고 11개 유형 중 어디 속하는지 분류**
2. **분류된 상권의 매출 강세·약세 카테고리 파악**
3. **새 트렌드 상품이 들어왔을 때 해당 매장에서 잘 팔릴지 판단**
4. **점주에게 "이 상권은 이런 특성이 있어요"라고 설명**

배율 해석:
- **1.0** = 12매장 평균
- **1.5+** = 강세 (평균보다 50% 더 팔림)
- **2.0+** = 압도적 (평균의 2배)
- **0.5 이하** = 약세
- **0.3 이하** = 거의 안 팔림

- 대용량 상품 (가족 나들이용)
- 지역 특산물 (여행·선물 목적)
- 캠핑 용품, 간단한 아웃도어 상품
- 계절 상품 (여름 아이스크림·음료, 겨울 핫팩·온음료)
- 분위기 있는 디저트·음료 (체류형 고객 대상)

### 약세 카테고리 (정성)
- 1인 간편식 도시락 (타겟 고객 적음)
- 저가 군것질 (학생 거의 없음)
- 즉석조리식 (오피스 대비 약함)

### 트렌드 추천 가이드
- ✅ 추천: 계절 특화 신상, 대용량·패밀리팩, 아웃도어·캠핑 콜라보, 지역 한정판
- ❌ 비추: 순수 SNS 트렌드·프리미엄 단품·1인 가구 간편식

### 상권 동학 (정성)
- 고객 프로필: 차량 이동 고객·가족 나들이·드라이브 고객 + 지역 주민 일부.
- 소비 동기: 대용량 구매·지역 특산물·캠핑 용품·계절 상품 — 목적 구매 중심.
- 시간대 패턴: 주말·주간 집중. 계절·날씨 영향 큼 (비 오면 매출 급감).
- 가격 감도: 중간 — 분위기·맛·서비스 중시, 가성비 절대 기준 아님.
- 반복 소비 앵커: 기본 편의 용품, 지역 특산물, 계절 음료.
- 트렌드 감응도: 낮음 — 순수 SNS 트렌드 약함. 계절 특화·분위기 상품에만 반응.

# 운영 시 주의사항 (AI에게)

## 1. 매장 분류 신호 우선순위
1. **위치 기반** (카카오 지도 API): 좌표 → 좌표 자체로 단지명/대학교명 매칭
2. **점포 분포** (소상공인 API): 반경 500m 카테고리 비중
3. **POS 데이터** (점주 가맹 후): 매출 패턴으로 최종 확정·보정

## 2. 한 매장 = 여러 상권태그 동시 가능
- 신촌르메이에르 = 대학가 + 유흥 + 역세권 (3개 동시)
- 만촌힐스테이트 = 학원가 + 일반주거 + 역세권 (3개 동시)
- 매칭 점수는 각 태그별 가중치 합산

## 3. 신뢰도 단서
- **카테고리 단위 패턴 (대분류)**: 신뢰 가능 (n=수만 거래)
- **개별 트렌드 상품 검증**: 표본 작음, 방향성만
- **시간대 분석**: CU 데이터에선 timestamp 결함으로 불가
- **시즌·요일 패턴**: 신뢰 가능

## 4. B2B 매장 식별 (반드시 분리)
- 단일 거래 100개+ 비중 5%+
- 명절 시즌 매출이 평소의 3배+
- 화·목 같은 특정 요일 집중
- → **일반 점포 패턴으로 보지 말 것**

## 5. "추정"과 "실측" 구분
- 본사B2B/명절B2B/근린B2B = **CU 12매장 중 각 1개씩만 표본** → 패턴은 명확하지만 일반화 주의
- 고급주거·학원가·대학가 = **각 1매장 표본** → 다른 매장 확보로 검증 필요
- 외곽주거·신도시 = **각 1매장 표본**
- 일반주거·로컬번화·가두상권 = **각 1매장 표본**
- → **표본 1매장 한계 항상 표기**

## 6. 트렌드 점수 계산 시
```
종합 점수 = 화제성×35% + 수명×30% + 적합도×35%
적합도 = 20(기본) + 매장상권태그 × 상품카테고리·속성 매칭 점수
```

## 7. 발주 개수는 추천 X
- 매장 캐파(매대 크기·재고 회전·자금)를 모름
- 방향성("강력 추천 - 충분히 발주" / "스킵")만 제시
- 정확한 개수는 점주 판단
